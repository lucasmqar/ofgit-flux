
# ofgit-flux

Aplicação FLUX (Vite + React + TypeScript) com build mobile via Capacitor.

## Rodar local (web)

```bash
npm install
npm run dev
```

## Build (web)

```bash
npm run build
```

## Android (Capacitor)

```bash
npx cap sync android
npx cap run android
```

## Variáveis de ambiente

Os arquivos `.env` e `.env.*` não são versionados.
